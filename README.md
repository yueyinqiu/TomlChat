# TomlChat: Save and Manage your Conversation with AI in a TOML File

1. Install [.NET](https://dotnet.microsoft.com/zh-cn/download)
2. Download `TomlChat.zip` in our release page
3. Unzip it
4. Run `./TomlChat.exe ./test.toml`
   1. A `test.toml` will be created
5. Edit the `test.toml`
6. Run `./TomlChat.exe ./test.toml` again to generate response
